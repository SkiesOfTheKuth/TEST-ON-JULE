# Documentation

This documentation provides a detailed overview of the Python Starter Project.

## Project Structure

The project is organized as follows:

- `src/`: Contains the source code for the project.
  - `python_starter_project/`: The main package for the project.
    - `__init__.py`: Initializes the package.
    - `math.py`: Contains math-related functions.
- `tests/`: Contains the tests for the project.
- `docs/`: Contains the project documentation.
- `README.md`: A general overview of the project.
- `CHANGELOG.md`: A log of changes to the project.
- `CONTRIBUTING.md`: Guidelines for contributing to the project.
- `LICENSE`: The license for the project.
- `pyproject.toml`: The project configuration file.

## Modules

### `python_starter_project.math`

This module contains functions for performing mathematical operations.

#### `add(a: int, b: int) -> int`

Adds two integers together.

- **Parameters:**
  - `a`: The first integer.
  - `b`: The second integer.
- **Returns:** The sum of the two integers.
