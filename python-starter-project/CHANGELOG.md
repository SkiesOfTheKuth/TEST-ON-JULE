# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2024-09-26
### Added
- Production readiness scaffolding (Makefile, Poetry tasks, coverage settings).
- GitHub Actions CI workflow with linting, type checks, and tests.
- Dockerfile with healthcheck and accompanying `.dockerignore`.
- Structured logging and configuration modules with tests.
- Documentation suite: architecture, operations, security, observability, migrations, onboarding.
- Security policy and responsible disclosure issue template.
- Pre-commit configuration and `.env` sample.

## [0.1.0] - 2024-09-26
### Added
- Initial project structure.
- `add` function in `python_starter_project.math`.
- `README.md`, `CHANGELOG.md`, `CONTRIBUTING.md`, and `LICENSE` files.
- Basic documentation in the `docs` folder.
